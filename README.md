# SMPLify-X_Modify

Modified SMPLify-X.
1. Add perspective Camera.
2. Support a video as input.
3. Support a 3D joint as initialization.
4. Support IK solver.
5. Add ground plane support.


# Usage

## IK solver for body and hands.

```./run_3D_joints.sh```

## Run a single video for single person.

``` temp.sh ```
